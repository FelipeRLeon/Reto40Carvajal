Se utiliza vue y vuetify como frameworks para el desarrollo del frontend.

1.Administrador: Puede ver todos los productos en el stock, ademas se encarga de crear, editar y eliminar los productos existentes en el stock de la tienda.

2.Cliente: puedever los productos en el stock con toda su informacion correspondiente, ademas puede a√±adir productos a una wish list.

Para iniciar el frontend ejecutar en la terminal de la carpeta frintend el comando "npm run serve"
